🎯 Guess the Number – Python Project

Guess the Number is a Python-based interactive game where the player tries to guess a randomly generated number within a specified range. The game provides hints if the guess is too high or too low. It’s a beginner-friendly project to learn Python logic, loops, and conditionals.

🚀 Features

🔢 Random number generation

💡 Hints for high or low guesses

🎮 Keeps track of attempts

🐍 Simple, beginner-friendly Python code

Optional GUI version with Tkinter

🛠️ Technologies Used

Python 3

random module

Standard input/output

Tkinter (optional for GUI)

⚙️ Installation

Clone the repository

git clone https://github.com/your-username/Guess-the-Number.git


Navigate to the project directory

cd Guess-the-Number


Install dependencies (if any)

pip install -r requirements.txt

▶️ How to Run
python main.py


The computer generates a random number

Enter your guesses one by one

The program provides hints: “Too High” or “Too Low”

Game continues until the correct number is guessed

Total attempts are displayed at the end

📌 How It Works

Computer generates a random number

User inputs guesses via console

Program compares input with the number

Provides feedback and counts attempts

Ends game when user guesses correctly
